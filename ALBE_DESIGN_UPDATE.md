## Customization

Want to change the text? Edit `index.html`:

**Line 290** - Change headline:
```html
<span class="line2">YOUR TEXT</span>
```

**Line 294** - Change description:
```html
<p>
    Your custom description here...
</p>
```

**Line 301** - Change button text:
```html
<a href="chat.html" class="btn-primary">YOUR BUTTON TEXT</a>
```
